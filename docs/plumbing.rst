Plumbing: Rescale HTC REST API functions
========================================

The api module contains functions that map 1:1 to the Rescale HTC API. They are
used as low-level plumbing for the library. The end user can use them directly
if desired, but often there is a more convenient function to use in the higher
level abstractions
